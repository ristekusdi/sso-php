# Pendahuluan

SSO Laravel oleh RistekUSDI adalah sebuah pustaka berbasis framework Laravel dan platform Keycloak yang digunakan untuk otentikasi pengguna pada aplikasi internal Universitas Udayana.

Pustaka ini terbagi menjadi 3 versi antara lain:

- Versi 1.x untuk Laravel versi 6.x ke atas.
- Versi 0.3.x untuk Laravel versi 5.5 hingga 5.8.
- Versi 0.2.x untuk Laravel versi 5.2 hingga 5.4.
- Versi 0.1.x untuk Laravel versi 5.1.

## Catatan

Pustaka ini berasal dari repositori GitHub [mariovalney/laravel-keycloak-web-guard](https://github.com/mariovalney/laravel-keycloak-web-guard).

Kami menggunakan kode tersebut untuk dimodifikasi sesuai kebutuhan internal kami. Kami ucapkan terima kasih kepada pengembang dari pustaka tersebut.
